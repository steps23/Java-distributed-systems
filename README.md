# Java Distributed Systems Assignments

Collection of four Java-based assignments developed in **September 2024** for a Distributed Systems course.  
The repository gathers independent projects related to message passing, distributed coordination, fault handling, global state capture, and simulation.

## Overview

This repository contains four separate assignments:

1. **Assignment 1** – TCP coordination and UDP multicast messaging with simulated packet loss and resend handling.
2. **Assignment 2** – Distributed coordination with barrier synchronization, centralized mutual exclusion, heartbeat-based failure detection, and Bully election.
3. **Assignment 3** – Chandy–Lamport distributed snapshot over a TCP-connected network of nodes.
4. **Assignment 4** – Building evacuation simulation in Java Swing using flocking-based agent movement, obstacles, exits, queues, and attraction points.

The repository also includes generated Javadoc documentation, compiled `.class` files, and a few operating-system metadata files.
